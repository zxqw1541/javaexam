명령> help
[사용법]
명령

[명령]
menu         메뉴 목록을 출력한다.
menu 번호     해당 번호의 메뉴로 간다. 
quit         프로그램을 종료한다.
help         도움말을 출력한다.

예) 

명령> menu
1 회원 관리
2 프로젝트 관리

명령>menu 1

회원관리> 

회원관리> help
list :    학생 목록을 리턴한다.
delete :  회원을 삭제하는 명령어
add  :    회원을 추가한다.

명령>menu 2

프로젝트관리> help
list :    프로젝트 목록을 리턴한다.
delete :  프로젝트를 삭제하는 명령어
add  :    프로젝트를 추가한다.
