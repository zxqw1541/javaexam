list : 프로젝트 목록을 리턴한다.
예)
번호  프로젝트명       시작일         종료일        멤버
1    쿠폰관리시스템    2015-12-23   2016-02-28   홍길동,임꺽정,김구
2    aaaa          2015-12-23   2016-02-28   김원봉,안중근,윤봉길


add : 프로젝트 정보를 추가하는 명령어
프로젝트명? 쿠폰관리시스템
시작일? 2015-12-23
종료일? 2016-02-28
멤버? 홍길동,임꺽정,김구
정말 저장하시겠습니까?(y/n) y
저장되었습니다.
정말 저장하시겠습니까?(y/n) n
취소하였습니다.

delete : 프로젝트를 삭제하는 명령어
프로젝트 번호? 1
정말 삭제하시겠습니까?(y/n) y
삭제 되었습니다.
정말 삭제하시겠습니까?(y/n) n
취소하였습니다.
